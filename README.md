# Personal Portfolio Website

A clean, modern, static portfolio website built using only HTML5 and CSS3. No JavaScript or frameworks were used, ensuring a lightweight and fast-loading experience.

## Features
- **Responsive Design**: Mobile-first approach compatible with all devices.
- **Clean UI/UX**: Professional Blue & Gray color theme with smooth hover effects.
- **Pure CSS**: No external libraries or frameworks (Bootstrap, Tailwind, etc.).
- **PWA Ready**: Includes `manifest.json` and basic PWA setup structure.
- **Pages Included**: Home, About, Skills, Projects, and Contact.

## Folder Structure
```
portfolio-website/
├── css/                # Global styles
├── assets/             # Images and Icons
├── index.html          # Home Page
├── about.html          # About Me
├── skills.html         # Technical Skills
├── projects.html       # Project Case Studies
├── contact.html        # Contact Info & Form
├── manifest.json       # PWA Manifest
└── offline.html        # Offline Fallback
```

## How to Run
1.  Unzip the folder.
2.  Open `index.html` in any modern web browser (Chrome, Firefox, Edge, Safari).

## Customization
- **Images**: Place your profile and project images in `assets/images/`.
- **Text**: Edit the HTML files to update your personal information.
- **Colors**: Modify the `:root` variables in `css/style.css` to change the color theme.
